HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML stands for Hyper Text Markup Language
  • It works as the building blocks of a webpage with HTML elements.
  • HTML documents are files that end with a .html extension.

CSS

  • CSS stands for Cascading Style Sheets.
  • It is used to describe the presentation of a document.
  • CSS is a cornerstone technology of the World Wide Web, along with HTML and JavaScript.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An if statement is a way of introducing decision-making into our code.
  • Functions are the building blocks of JavaScript. Their job is to perform a specific task.
  • Arrays are a group of data stored in a single variable. To access a single type of data, you must use an index or location number.
  • The for Loop uses predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.

What I Learned

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Class indicators are marked with a period.
